# 🧩 Chrome Extension (Chrext)

A powerful Chrome extension featuring Razorpay integration and backend services.

---

## 🌟 Overview

A full-stack browser extension with payment gateway integration, backend API services, and popup interface.

## ✨ Features

- **Payment Integration** - Razorpay support
- **Backend Service** - Node.js/Python backend
- **Popup UI** - Interactive extension interface
- **Secure** - API key management

## 📁 Project Structure

```
chrext/
├── backend/         # Server-side code
├── popup/           # Extension UI
├── content-scripts/ # Page interaction
├── background/      # Background workers
└── manifest.json    # Extension config
```

## 🚀 Quick Start

1. Load unpacked in `chrome://extensions`
2. Select `chrext` folder
